# Dr. Cabello — WordPress Theme

Tema de WordPress para el proyecto Dr.Cabello. Contiene las plantillas, estilos y activos usados en el sitio.

Contenido principal:
- Archivos PHP del tema (`index.php`, `front-page.php`, `header.php`, `footer.php`, `functions.php`, etc.)
- Carpeta `assets/` con CSS, JS, imágenes y video
- `template-parts/` con secciones reutilizables

Instrucciones rápidas
1. Copia esta carpeta a `wp-content/themes/` en tu instalación de WordPress.
2. Activa el tema desde el panel de administración de WordPress.
